Preguntas y respuestas

    
1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

      Java está más orientado a aplicaciones empresariales y Android. Es compilado, de tipado estático y se ejecuta en la JVM. JavaScript es interpretado, de tipado dinámico y esencial para el desarrollo web frontend y backend con Node.js. Ambos están orientados a objetos y su sintaxis se basa en C.
    

    
2. ¿Qué métodos tiene el objeto Date? (al menos 5)

      getFullYear(), getMonth(), getDate(), getHours() y getMinutes().
    

    
3. ¿Qué métodos tienen los arreglos? (al menos 5)

      push(), pop(), shift(), map() y filter().
    

    
4. ¿Cómo se declara una variable con alcance local dentro de una función?

      Con let o const dentro del bloque de la función.
    

    
5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

      Pueden leerse y modificarse desde cualquier parte, lo que puede causar colisiones de nombres, efectos secundarios no deseados y complicar depuración, mantenimiento y uso de memoria.
